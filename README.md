# Dynamic Dex

based on [Lhea's Porydex](https://github.com/lhearachel/porydex)

## Features for Players
The following info is automatically generated when using the companion tools for Dynamic Dex on top of the original porydex. 

### Items
- Shows changes from vanilla descriptions
- Shows locations where found and obtainment methods
- Gen 4 Methods include: Ground pickup, NPC gifts, Wild Held Items
- Gen 5 Methods include: Ground pickup, commonscript trainer rewards, marts

- Now supports any format for wild encounter table names, just specify in the `rates` property
- Can import encounters if using Dynamic Calc or by manual clicking encounters to mark as caught
- Displayed moveset of wild encounters at the level they appear at
- Takes into account dupes to show updated encounter rates for nuzlockers

## Feature for Romhack Developers

Load any Gen 3 Romhack by uploading the .gba file and .toml file from HexManiac Advance
Load any Gen 4 Romhack by uploading the .nds file 

Once a rom has been loaded, you can view a local Damage Calculator from the More Tab or download the raw javascript calc/dex exported data

<img width="314" height="299" alt="Screenshot 2026-05-03 at 1 34 45 PM" src="https://github.com/user-attachments/assets/2d7aed5b-fa54-4eac-93fb-14ddba4aa697" />

If you are done developing your romhack and want to permanently host your Calc/Dex data so that users can view it without having to upload a rom, contact hzla on discord.
